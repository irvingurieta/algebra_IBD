Extensión: el límite de la condición “grado 2 o 3”

Analiza g(x)=x⁴+4. No tiene ceros racionales, pero es reducible sobre ℚ:

x⁴ + 4 = (x² − 2x + 2)(x² + 2x + 2).
Este ejemplo muestra que el criterio “reducible ⇔ tiene cero racional” no se extiende automáticamente a grado 4.
